Real-time Listeners

Keep your data in sync across devices

üëÇ Collection Listeners
Watch entire collections for changes:
const unsubscribe = onSnapshot( collection(db, "users"), (snapshot) => { snapshot.docChanges().forEach((change) => { if (change.type === "added") { console.log("New:", change.doc.data()); } else if (change.type === "modified") { console.log("Modified:", change.doc.data()); } else if (change.type === "removed") { console.log("Removed:", change.doc.data()); } }); } ); // Stop listening unsubscribe();
üìÑ Document Listeners
Listen to a single document for updates:
const unsubscribe = onSnapshot( doc(db, "users", "user123"), (docSnapshot) => { if (docSnapshot.exists()) { console.log("User data:", docSnapshot.data()); } else { console.log("Document deleted"); } } );
üí° Attach error handler as second callback
üîÑ Query Listeners
Listen to filtered queries with real-time updates:
const q = query( collection(db, "users"), where("status", "==", "active") ); const unsubscribe = onSnapshot(q, (snapshot) => { console.log("Active users:", snapshot.docs); });
üö´ Cleanup & Unsubscribe
Always stop listening when done:
useEffect(() => { const unsubscribe = onSnapshot( doc(db, "users", userId), (doc) => { setUser(doc.data()); } ); // Cleanup: stop listening on unmount return () => unsubscribe(); }, [userId]);

üì± Offline Persistence
Enable offline caching for web/mobile apps:
import { initializeFirestore, enableIndexedDbPersistence } from "firebase/firestore"; const db = initializeFirestore(app, {}); enableIndexedDbPersistence(db) .catch((err) => { if (err.code === "failed-precondition") { // Multiple tabs open } else if (err.code === "unimplemented") { // Browser not supported } });
Benefits: App works offline, syncs when reconnected
