üì¶ Batch Transactions
Perform multiple operations in one atomic batch:
const batch = writeBatch(db); // Multiple operations batch.set(doc1Ref, { value: 1 }); batch.update(doc2Ref, { counter: 5 }); batch.delete(doc3Ref); // All or nothing await batch.commit();
Limits: Max 500 writes per batch, 10 MiB total size
